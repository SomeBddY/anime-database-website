NodeJS-Express website
======================

Kickstarter for NodeJS Express Website

------------------------
-   Hyper
-   Atom

### Back-End:
-   NodeJS
-   Express
-   MongoDB

### Front-End:
-   JavaScript
-   HTML
-   CSS
-   Bootstrap
-   jQuery
-   EJS (template)

Main idea of the project:
-------------------------
-   Create, read, update and delete data

-   Authentication
